Other Interesting things I've done

Worked on a Fuzzy String Search Project

I used an implementation of the Levenshtein distance to fuzzily match names against a list of names. Whenever I did not have a 100% match, I looked at the top 3 matches to figure out the best match. Sometimes, the second – best match was the correct match. I used the fuzzywuzzy package on Python to do the match. The project required some data cleaning and formatting, so that the search was made easier.

Web Scraping

For a project I’m currently working on, I needed to scrape lyrics off the Internet. I used lyrics.com for the scraping. I did this in two parts: the first was getting all the links of the artist names, each link will have all the songs by that artist. The next part is using those links to get all the lyrics. I used BeautifulSoup and Selenium packages on Python for this task.

During the summer break I came across mathematical art: Fractals. They are geometrical figures, with fractional dimensions and whose parts have the same characteristics as the whole figure. I plotted the Sierpinski Triangle, the Koch Snowflake and the Mandelbrot Set in quadradic and cubic.
